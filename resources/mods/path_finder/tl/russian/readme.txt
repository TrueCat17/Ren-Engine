[Общее]
Отметьте на локациях точки для посещения.

После нажмите "Старт".
Персонаж пройдёт от начальной до конечной точки через все промежуточные места.


[brute_force]
Параметр <brute_force> определяет необходимость пытаться искать кратчайший путь между точками одной локации через выход из неё.


[Выбор точки]
Вы можете отметить точку на доступном для передвижения (затенённом) месте локации.
Вы можете выбрать локацию на главном экране.
Используйте кнопки WASD для перемещения по локации.
Нажмите <Esc> для выхода из экрана локации в главный экран.
